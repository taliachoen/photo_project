Photo Project 📸

Welcome to Photo Project!
A web-based application designed to showcase and manage a collection of photographs. This project features a responsive photo gallery with modern design elements to enhance user experience.

🎯 Features

Responsive Layout: Optimized for all screen sizes.

Hover Effects: Interactive hover animations for an engaging interface.

Clean Folder Structure: Easy to navigate and maintain.

🛠️ Technologies Used

HTML5: To structure the photo gallery.

CSS3: For styling and creating visual appeal.

🚀 Getting Started

Follow these steps to set up and view the project on your local machine:

Prerequisites

A web browser (e.g., Chrome, Firefox, Safari).

Installation

Clone the repository:

git clone https://github.com/taliachoen/photo_project.git

Navigate to the project directory:

cd photo_project

Open the index.html file in your preferred web browser to explore the gallery.

images/: Directory to store all photographs displayed in the gallery.

index.html: The main HTML file that structures the web page.
